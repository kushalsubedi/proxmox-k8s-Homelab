# Screenshots

Place setup screenshots here, named to match the document/step they illustrate, e.g.:

- `01-proxmox-installer-disk-select.png`
- `02-vmbr0-web-ui.png`
- `07-kubeadm-init-output.png`
- `09-cilium-status.png`
- `11-macos-finder-smb-share.png`

Keeping a consistent `NN-description.png` naming convention (matching the `docs/NN-*.md` numbering) makes it easy to cross-reference which screenshot belongs to which step.
