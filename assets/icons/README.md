# Icons

Place any custom icon assets used by the `diagrams/*.drawio` files here (e.g. Proxmox, Kubernetes, Cilium, Samba logos used as diagram node shapes).

draw.io supports referencing external images as custom shapes — if you replace the built-in shape styles in the `.drawio` files with image-based shapes, store the source images here and reference them with a relative path so the diagrams remain portable within this repository.
